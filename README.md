# vue-reaction-timer

A reaction timer web app made using Vue.js.

[Ready to test your reflexes?](https://ezzylan.github.io/vue-reaction-timer/)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
